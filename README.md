# 3d-Engine
В этом проекте мне удалось создать движок для отображения трёхмерных объектов. Туда встроен шар, тор, куб и загружаемый пользователем объект. Пока программа не оптимизирована до конца, но текущей оптимизации numba, jit, numpy хватает для стабильной работы при неэкстремальных нагрузках. В дальнейшем программа будет переписана на C++.

На данный момент в программе при помощи специального окна можно выбрать любой из трёх встроенных примитивов: шар, куб, тор. Также во вкладке "Файл" можно загрузить 3d-модель в одном из 2 форматов: obj и stl. Программа пока плохо справляется с большим числом полигонов и будет сильно лагать и зависать. Во вкладке "Справка" можно будет посмотреть краткую техническую информацию о программе.

Для работы программы требуются библиотеки. Их необходимо установить отдельно.

Для примитива куб можно выбррать один из предложенных цветов, вернуть к стартовой позиции и остановить вращение. По умолчанию куб вращается. Также можно нажатием кнопки "Стоп" полностью остановить вращение. При вращении куба он мышкой он полностью останавливается, для возобновления вращения необходимо нажать кнопку "Стоп".

Для примитивов тор и сфера единственным отличием будет возможность выбора детализации (числа полигонов). Всё остальное будет как в кубе.

Для пользовательского файла будут справедливы настройки куба, но без возможности выбора цвета.

Также во вкладке "Файл" есть кнопка выхода из программы.
